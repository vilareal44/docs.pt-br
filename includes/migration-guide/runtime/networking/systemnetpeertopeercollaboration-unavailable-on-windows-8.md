### <a name="systemnetpeertopeercollaboration-unavailable-on-windows-8"></a>System.Net.PeerToPeer.Collaboration não disponível no Windows 8

#### <a name="details"></a>Detalhes

#### <a name="suggestion"></a>Sugestão
