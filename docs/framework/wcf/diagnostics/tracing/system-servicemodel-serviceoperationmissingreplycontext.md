# <a name="systemservicemodelserviceoperationmissingreplycontext"></a>System.ServiceModel.ServiceOperationMissingReplyContext
System.ServiceModel.ServiceOperationMissingReplyContext  
  
## <a name="description"></a>Descrição  
 ReplyContext é nulo para uma operação bidirecional. Os dados de rastreamento exibem a seguinte mensagem: "a operação de resposta não tem IRequestContext para usar para a resposta".  
  
 Esse rastreamento é útil para depuração durante o desenvolvimento.
