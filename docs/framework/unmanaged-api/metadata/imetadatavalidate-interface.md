# <a name="imetadatavalidate-interface"></a>Interface IMetaDataValidate
Fornece métodos para validar assinaturas de metadados.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método ValidateMetaData](imetadatavalidate-validatemetadata-method.md)|Valida as assinaturas de metadados dos objetos no escopo de metadados atual.|  
|[Método ValidatorInit](imetadatavalidate-validatorinit-method.md)|Define um sinalizador que especifica o tipo do módulo no escopo de metadados atual e registra o método de retorno de chamada especificado para erros de validação.|
