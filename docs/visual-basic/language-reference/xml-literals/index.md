# <a name="xml-literals-visual-basic"></a>Literais XML (Visual Basic)
Os tópicos nesta seção documentam a sintaxe de literais XML no Visual Basic. A sintaxe de literal do XML permite incorporar XML diretamente ao código.

|Tópico|Descrição|  
|-----------|-----------------|  
|[Literal do Elemento XML](xml-element-literal.md)|Descreve a sintaxe de literais que representam objetos <xref:System.Xml.Linq.XElement>.|  
|[Literal de Documento XML](xml-document-literal.md)|Descreve a sintaxe de literais que representam objetos <xref:System.Xml.Linq.XDocument>.|  
|[Literal CDATA XML](xml-cdata-literal.md)|Descreve a sintaxe de literais que representam objetos <xref:System.Xml.Linq.XCData>.|  
|[Literal de Comentário XML](xml-comment-literal.md)|Descreve a sintaxe de literais que representam objetos <xref:System.Xml.Linq.XComment>.|  
|[Literal de Instrução de Processamento XML](xml-processing-instruction-literal.md)|Descreve a sintaxe de literais que representam objetos <xref:System.Xml.Linq.XProcessingInstruction>.|
