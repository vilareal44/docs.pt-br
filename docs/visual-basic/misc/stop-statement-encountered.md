# <a name="stop-statement-encountered"></a>Instrução stop encontrada
Uma `Stop` instrução, que suspende a execução, foi encontrada.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1. Remova a `Stop` instrução.
