# <a name="xml-name-expected"></a>Nome XML esperado
Uma propriedade XML literal ou XML AXIS foi especificada sem um nome.  
  
 **ID do erro:** BC31146  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicione um nome para a propriedade XML literal ou XML Axis.
