# <a name="end-operator-must-be-the-first-statement-on-a-line"></a>'End Operator' deve ser a primeira instrução em uma linha
Uma `End Operator` instrução segue um separador de instruções de dois-pontos ( `:` ). `End Operator`deve ser a única instrução em sua linha de origem.  
  
 **ID do erro:** BC33006  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Separe as várias instruções em linhas diferentes.
