# <a name="inherits-statements-must-precede-all-declarations-in-an-interface"></a>Instruções 'Inherits' devem preceder todas as declarações em uma interface
Uma `Inherits` instrução deve ser a primeira linha que não seja em branco e não de comentário em uma definição de classe.  
  
 **ID do erro:** BC30357  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Mova sua `Inherits` instrução para que ela preceda todas as declarações.
